mailcow is a full-featured, open-source mail server stack that runs entirely in Docker. It bundles everything you need — SMTP, IMAP, antispam, antivirus, webmail, and a REST API — into a single `docker compose up` deployment.

## Get started in 4 steps

<Steps>
  <Step title="Check prerequisites">
    Ensure your server meets the [system requirements](/prerequisites) and DNS records are configured correctly.
  </Step>
  <Step title="Clone and configure">
    Clone the repository and run `generate_config.sh` to create `mailcow.conf` with your hostname and timezone.
  </Step>
  <Step title="Start the stack">
    Run `docker compose up -d` to pull images and start all services. The web UI is available at `https://your-hostname` within a few minutes.
  </Step>
  <Step title="Add your first domain">
    Log in to the admin UI with `admin` / `moohoo`, then [add a domain and create mailboxes](/administration/domains-mailboxes).
  </Step>
</Steps>

<Note>
  The default admin password is `moohoo`. Change it immediately after your first login.
</Note>
